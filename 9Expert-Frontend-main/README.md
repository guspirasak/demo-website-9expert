## Installation

[![URL][URL]][URL-URL]

### 1. Install dependencies

```bash
npm install
```

### 2. Run dev environment

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy and Production

1. Push update code to Github repository.
2. Merge update code to main branch.
3. Check deploy status in Vercel.
